## [0.1.0] - 2025-01-01

### Added
- Initial release of r3f-gltf-outlines
- `OutlineEffect` component for React Three Fiber
- Post-processing outline shader based on depth, normals, and surface IDs
- Support for customizable outline colors and parameters
- Debug visualization modes for fine-tuning
- TypeScript support with full type definitions
- ESM and CJS builds
- Comprehensive documentation and examples
- Unit tests for core utilities
- CI/CD pipeline with GitHub Actions

### Features
- High-quality outline rendering using webgl-outlines technique
- Configurable depth and normal-based edge detection
- Surface ID computation for improved outline quality
- Vertex welding utility for mesh optimization
- FXAA anti-aliasing pass
- Real-time parameter adjustment
- Multiple debug visualization modes
